## LangSmith Updates

📦 **LangSmith Sandboxes** now offers a free trial, so you can give your agents as secure, isolated environment to run code, test changes, and complete long-running work safely. [Read Sandboxes docs →](https://docs.langchain.com/langsmith/sandboxes)

💬 **LangSmith Fleet:** Bring agents into Slack with one click. Tag agents in channels, handle follow-ups and HITL approvals in threads, send and receive documents, and more. [Read announcement →](https://www.langchain.com/blog/new-in-langsmith-fleet-bring-agents-into-slack-in-one-click)

🗣️ **Voice agent tracing in LangSmith:** Capture audio, STT and TTS latency, interruptions, tool calls, and more in one trace for agents built with Pipecat, LiveKit, OpenAI Realtime, and Gemini Live. [Learn more →](https://docs.langchain.com/langsmith/integrations)

## Open Source

🧠 **OpenWiki Brains:** A general-purpose memory layer that turns sources like Gmail, Notion, git repos, X, Hacker News, and web search into a local wiki your agents can use and keep updated automatically. [Read the blog →](https://www.langchain.com/blog/introducing-openwiki-brains-general-purpose-wiki-memory-for-agents)

🔎 **Deep Agents and Harbor** now provide a unified stack for evaluating agents, with isolated sandbox runs, parallel trials, and LangSmith traces that show why each agent succeeds or fails. [Learn more →](https://www.langchain.com/blog/unified-stack-for-evaluating-agents)

🧩 **RLMs in Deep Agents:** Use dynamic subagents to break large-context tasks into programmatic subagent calls, helping agents reduce context rot and orchestrate work over large inputs. [Read guide →](https://www.langchain.com/blog/how-to-use-rlms-in-deep-agents)
