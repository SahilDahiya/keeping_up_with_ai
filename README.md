# keeping_up_with_ai

Scrape AI engineering blogs into a topic-organized markdown knowledge base.

Codex users: start with `AGENTS.md`. Claude Code users can also use
`.claude/skills/organize-kb/SKILL.md`.

The KB is intentionally selective: archive pages, promotions, low-detail release
notes, generic company news, and off-topic posts should be skipped from `_inbox/`
instead of filed.
